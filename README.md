# ココロジック・カフェ

「ココロジック・カフェ」は、物語体験を中心としたカフェ運営ゲームです。プレイヤーは自身の体験から生まれた疑問を、カフェ運営を通じて検証し、自分だけの「発見ノート」を完成させていきます。プレイヤーの主体的な気づきと、その過程の物語体験を最優先する「体験先行・仮説検証型」のゲームです。

## 開発

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### テスト

```bash
npm run test
```
