## 第一部：問いと仮説、最初の発見

### Chapter 1: 三つの選択肢

| 項目               | 内容                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | 序章クリア                                                                                                                                                                                                                                                                                                                                                                                           |
| **発見される概念** | **おとり効果 (Decoy Effect)**                                                                                                                                                                                                                                                                                                                                                                        |
| **パート1：体験**  | **状況:** 新しいコーヒー豆を仕入れる。<br>**イベント:** 行商人がプレイヤーに選択肢を提示する。<br> 1. 「A: スタンダード豆 (50g, 500円)」<br> 2. 「B: プレミアム豆 (100g, 1000円)」<br>少し悩んでいると、行商人がおとりの選択肢を追加する。<br> 3. 「C: 微妙な豆 (50g, 950円)」<br>プレイヤーはCの存在により、Bが非常にお得に感じて選択する。<br>**教授の問い:** 「君は、なぜあの豆を選んだのかね？」 |
| **パート2：応用**  | **目的:** 利益率の高い「プレミアムコーヒーセット」の売上を伸ばしたい。<br>**施策:** 既存の「スタンダードセット(500円)」「プレミアムセット(1000円)」のメニューに、おとりとして「リッチセット(950円、内容はスタンダードとほぼ同じ)」を追加する。<br>**結果:** プレミアムセットの注文率が上昇し、1日の売上が目標額を達成する。                                                                          |
| **発見ノート記録** | **体験したこと:** 仕入れの時、3つの選択肢を提示されて、つい真ん中のを選んでしまった。<br>**そこからの気づき:** 人は極端な選択肢を避ける傾向があるのかもしれない。<br>**やってみたこと:** 自分の店でも、3つの価格帯のメニューを作ってみた。<br>**わかったこと:** 真ん中の価格のメニューが一番売れるようになった。<br>**自分だけの法則:** 松竹梅の法則                                                 |
| **報酬**           | +50 評判ポイント、キーアイテム「アンティークなメニューブック」                                                                                                                                                                                                                                                                                                                                       |

### Chapter 2: 最初の印象

| 項目               | 内容                                                                                                                                                                                                                                                                                                                                                                                              |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | Chapter 1 クリア、評判が50以上                                                                                                                                                                                                                                                                                                                                                                    |
| **発見される概念** | **アンカリング効果 (Anchoring Effect)**                                                                                                                                                                                                                                                                                                                                                           |
| **パート1：体験**  | **状況:** カフェに飾る絵画を購入しに、画廊を訪れる。<br>**イベント:** 店主が最初に、非常に高価なアーティストの大型絵画（50,000円）を熱心に勧めてくる。プレイヤーは「高すぎる」と感じるが、その後に提示された手頃な価格の風景画（5,000円）が、なぜか非常に安く感じて購入してしまう。<br>**教授の問い:** 「最初にあの大きな絵を見ていなかったとしても、君は同じ値段でその風景画を買っていたかね？」 |
| **パート2：応用**  | **目的:** 新しく開発した「季節のフルーツタルト(800円)」の売上を伸ばす。<br>**施策:** メニューの一番上に、採算度外視の超高級「女王陛下のジュエリータルト(3,000円)」を数量限定で掲載する。このタルトは売れなくても良い。<br>**結果:** 「女王陛下のジュエリータルト」の価格がアンカーとなり、他のケーキ（特に「季節のフルーツタルト」）が手頃に見えるようになり、注文率が上昇する。                  |
| **発見ノート記録** | **体験したこと:** 画廊で最初にすごく高い絵を見たら、後の絵が安く感じて買ってしまった。<br>**そこからの気づき:** 最初に見た数字や情報が、後の判断の基準（アンカー）になるみたいだ。<br>**やってみたこと:** 自分の店のメニューに、すごく高いケーキを載せてみた。<br>**わかったこと:** 他のケーキがよく売れるようになった。<br>**自分だけの法則:** 見せ方の順番の法則                                |
| **報酬**           | +70 評判ポイント、内装アイテム「本日のおすすめ用イーゼル」                                                                                                                                                                                                                                                                                                                                        |

### Chapter 3: 言葉の額縁

| 項目               | 内容                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | Chapter 2 クリア、評判が150以上                                                                                                                                                                                                                                                                                                                                                                      |
| **発見される概念** | **フレーミング効果 (Framing Effect)**                                                                                                                                                                                                                                                                                                                                                                |
| **パート1：体験**  | **状況:** カフェの衛生管理のため、業者と契約する。<br>**イベント:** 業者から2つのプランを提示される。<br> 1. 「プランA: 顧客満足度90%の実績です！」<br> 2. 「プランB: クレーム率10%のリスクがあります」<br>両者は全く同じ内容だが、プレイヤーはポジティブな表現のプランAを選ぶ。<br>**教授の問い:** 「もし、二つのプランが全く同じものだったとしたら、君の選択の決め手は、一体何だったのだろうね？」 |
| **パート2：応用**  | **目的:** ヘルシー志向の顧客向けに開発した「豆乳ラテ」の売れ行きが悪い問題を解決する。<br>**施策:** 現在の「脂肪分10%」という表記を、「脂肪分90%カット」というポジティブな表現に変更する。<br>**結果:** 同じ商品であるにも関わらず、ポジティブな表現に変えたことで注文数が大幅に増加する。                                                                                                           |
| **発見ノート記録** | **体験したこと:** 同じ内容なのに「満足度90%」と「クレーム率10%」と書かれていると、印象が全然違って見えた。<br>**そこからの気づき:** 人は、同じ事実でも、伝え方（フレーム）によって受け取り方が変わる。<br>**やってみたこと:** 商品の説明を、ネガティブな表現からポジティブな表現に変えてみた。<br>**わかったこと:** 商品が以前より売れるようになった。<br>**自分だけの法則:** 言葉の額縁の法則       |
| **報酬**           | +100 評判ポイント、レシピ「言葉を飾るシュガー」                                                                                                                                                                                                                                                                                                                                                      |

## 第二部：すれ違う心と、雨の日のコーヒー

### Chapter 4: 今日の喜び

| 項目               | 内容                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **解放条件**       | 第二部開始                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **発見される概念** | **現在志向バイアス (Present Bias)**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **パート1：体験**  | **状況:** カフェの改装工事の契約を結ぶ。<br>**イベント:** 工務店の親方から2つの支払いプランを提示される。<br> 1. 「今すぐ契約金（5,000円）を払ってくれれば、総額から10,000円値引きしますよ」<br> 2. 「1ヶ月後に契約するなら、値引きはないです」<br>プレイヤーは将来の大きな利益（1ヶ月待てば5,000円得する）よりも、目先の損失回避（今払わないと10,000円の割引を逃す）と即時決定の安心感を選び、その場で契約してしまう。<br>**教授の問い:** 「未来の大きな虹を約束されるより、今日の小さな傘の方が、人の心を温めることもある。…君は、本当に『今日』決める必要があったのかね？」 |
| **パート2：応用**  | **目的:** 雨の日の客足を増やす。<br>**施策:** これまで効果のなかった「将来使えるポイント2倍」のような施策をやめ、「雨の日限定！ケーキもう一個プレゼント」のような、その場で即座に得られる喜びを提供する施策を試す。<br>**結果:** 雨の日の客数が増加し、売上が改善される。                                                                                                                                                                                                                                                                                                      |
| **発見ノート記録** | **体験したこと:** 改装工事の契約で、1ヶ月待てばもっと得だったのに、目先の割引に惹かれて焦って決めてしまった。<br>**そこからの気づき:** 人は、未来の大きなご褒美より、今の小さなご褒美を大切にしてしまうことがある。<br>**やってみたこと:** 雨の日に、将来の割引じゃなくて、その場ですぐもらえるプレゼントを提供してみた。<br>**わかったこと:** 雨の日でもお客さんが来てくれるようになった。<br>**自分だけの法則:** 今すぐもらえる喜びの法則                                                                                                                                    |
| **報酬**           | +120 評判ポイント、内装アイテム「雨音の聞こえる窓」                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
