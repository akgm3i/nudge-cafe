## 第一部：問いと仮説、最初の発見

### Chapter 1: 三つの選択肢

| 項目               | 内容 -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | 序章クリア -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **発見される概念** | **おとり効果 (Decoy Effect) - 非対称優位性** -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **パート1：体験**  | **状況:** 新しいコーヒー豆を仕入れる。<br>**イベント:** 行商人がプレイヤーに3つの選択肢を提示する。<br>・ **A (競合):** 品質70, 価格700円<br>・ **B (ターゲット):** 品質90, 価格1000円<br>・ **C (おとり):** 品質80, 価格1100円<br>**心理効果:** BとCを比較すると、Bの方が「品質が高く、かつ値段も安い」ため、Bが明らかに優れている（非対称優位）。この比較の容易さから、プレイヤーはAとBの比較（品質はBが高いが、値段も高い）を深く吟味することなく、Bが合理的で良い選択だと感じてしまう。<br>**教授の問い:** 「もしCの選択肢がなかったとしても、君はAとBの比較に同じくらい自信を持って、Bを選んでいただろうか？」 - |
| **パート2：応用**  | **目的:** 利益率の高い「プレミアムコーヒーセット」の売上を伸ばしたい。<br>**施策:** 既存の「スタンダードセット(品質70, 700円)」と「プレミアムセット(品質90, 1000円)」のメニューに、おとりとして「リッチセット(品質80, 1100円)」を追加する。<br>**結果:** プレミアムセットの注文率が上昇し、1日の売上が目標額を達成する。 -                                                                                                                                                                                                                                                                                            |
| **発見ノート記録** | **下書き(パート1完了時):**<br>・**体験したこと:** コーヒー豆の仕入れで3つの選択肢を提示され、品質と価格を比べた結果、一番お得に思えたBを選んだ。<br>・**そこからの気づき:** 明らかに劣る選択肢Cがあることで、BがAよりも魅力的に見えた。比較の仕方が判断に影響するのかもしれない。<br>**完成(パート2完了時):**<br>・**やってみたこと:** 自分の店のメニューでも、ターゲット(B)とおとり(C)を並べて、お客さんの選択を誘導してみた。<br>・**わかったこと:** 狙い通り、ターゲットであるプレミアムセットの注文が増えた。<br>・**自分だけの法則:** 見せかけの第三選択の法則                                                   |
| **報酬**           | +50 評判ポイント、キーアイテム「アンティークなメニューブック」 -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |

### Chapter 2: 最初の印象

| 項目               | 内容                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | Chapter 1 クリア、評判が50以上                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **発見される概念** | **アンカリング効果 (Anchoring Effect)**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **パート1：体験**  | **状況:** カフェに飾る絵画を購入しに、画廊を訪れる。<br>**イベント:** 店主が最初に、非常に高価なアーティストの大型絵画（50,000円）を熱心に勧めてくる。プレイヤーは「高すぎる」と感じるが、その後に提示された手頃な価格の風景画（5,000円）が、なぜか非常に安く感じて購入してしまう。<br>**教授の問い:** 「最初にあの大きな絵を見ていなかったとしても、君は同じ値段でその風景画を買っていたかね？」<br>**[デザインノート: このシナリオも、絵画の持つ「品質」や「希少性」といった具体的な属性を定義し、より巧妙な選択を設計する必要がある]** |
| **パート2：応用**  | **目的:** 新しく開発した「季節のフルーツタルト(800円)」の売上を伸ばす。<br>**施策:** メニューの一番上に、採算度外視の超高級「女王陛下のジュエリータルト(3,000円)」を数量限定で掲載する。このタルトは売れなくても良い。<br>**結果:** 「女王陛下のジュエリータルト」の価格がアンカーとなり、他のケーキ（特に「季節のフルーツタルト」）が手頃に見えるようになり、注文率が上昇する。                                                                                                                                                          |
| **発見ノート記録** | **下書き(パート1完了時):**<br>・**体験したこと:** 画廊で最初にすごく高い絵を見たら、後の絵が安く感じて買ってしまった。<br>・**そこからの気づき:** 最初に見た数字や情報が、後の判断の基準（アンカー）になるみたいだ。<br>**完成(パート2完了時):**<br>・**やってみたこと:** 自分の店のメニューに、すごく高いケーキを載せてみた。<br>・**わかったこと:** 他のケーキがよく売れるようになった。<br>・**自分だけの法則:** 見せ方の順番の法則                                                                                                    |
| **報酬**           | +70 評判ポイント、内装アイテム「本日のおすすめ用イーゼル」                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |

### Chapter 3: 言葉の額縁

| 項目               | 内容                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | Chapter 2 クリア、評判が150以上                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **発見される概念** | **フレーミング効果 (Framing Effect)**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **パート1：体験**  | **状況:** カフェの衛生管理のため、業者と契約する。<br>**イベント:** 業者から2つのプランを提示される。<br> 1. 「プランA: 顧客満足度90%の実績です！」<br> 2. 「プランB: クレーム率10%のリスクがあります」<br>両者は全く同じ内容だが、プレイヤーはポジティブな表現のプランAを選ぶ。<br>**教授の問い:** 「もし、二つのプランが全く同じものだったとしたら、君の選択の決め手は、一体何だったのだろうね？」<br>**[デザインノート: このシナリオも、プラン内容の「品質」などを属性として定義し、より巧妙な選択を設計する必要がある]** |
| **パート2：応用**  | **目的:** ヘルシー志向の顧客向けに開発した「豆乳ラテ」の売れ行きが悪い問題を解決する。<br>**施策:** 現在の「脂肪分10%」という表記を、「脂肪分90%カット」というポジティブな表現に変更する。<br>**結果:** 同じ商品であるにも関わらず、ポジティブな表現に変えたことで注文数が大幅に増加する。                                                                                                                                                                                                                                   |
| **発見ノート記録** | **下書き(パート1完了時):**<br>・**体験したこと:** 同じ内容なのに「満足度90%」と「クレーム率10%」と書かれていると、印象が全然違って見えた。<br>・**そこからの気づき:** 人は、同じ事実でも、伝え方（フレーム）によって受け取り方が変わる。<br>**完成(パート2完了時):**<br>・**やってみたこと:** 商品の説明を、ネガティブな表現からポジティブな表現に変えてみた。<br>・**わかったこと:** 商品が以前より売れるようになった。<br>・**自分だけの法則:** 言葉の額縁の法則                                                           |
| **報酬**           | +100 評判ポイント、レシピ「言葉を飾るシュガー」                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

## 第二部：すれ違う心と、雨の日のコーヒー

### Chapter 4: 今日の喜び

| 項目               | 内容 -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**       | 第二部開始 -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **発見される概念** | **現在志向バイアス (Present Bias)** -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **パート1：体験**  | **状況:** 新しい紅茶ブランドの販促で、営業担当者が無料サンプルを提供している。<br>**イベント:** 担当者がプレイヤーに2つの選択肢を提示する。<br> 1. 「A: **スタンダード・ティーボックス（1,000円相当）**を、**今すぐ**お渡しします」<br> 2. 「B: **プレミアム・ティーボックス（2,500円相当）**を、**来週**お渡しします」<br>プレイヤーは客観的には価値が高いと分かっているBよりも、目の前の確実な報酬であるAを選んでしまう。<br>**教授の問い:** 「今日の小さな喜びは、明日の大きな虹よりも、重く感じられることがある。君は、あの時どちらが『本当に』欲しかったのだろうね？」 - |
| **パート2：応用**  | **目的:** 雨の日の客足を増やす。<br>**施策:** これまで効果のなかった「将来使えるポイント2倍」のような施策をやめ、「雨の日限定！ケーキもう一個プレゼント」のような、その場で即座に得られる喜びを提供する施-策を試す。<br>**結果:** 雨の日の客数が増加し、売上が改善される。 -                                                                                                                                                                                                                                                                                                  |
| **発見ノート記録** | **下書き(パート1完了時):**<br>・**体験したこと:** 無料サンプルをもらう時、来週もらえる価値の高いものより、今すぐもらえる方を選んでしまった。<br>・**そこからの気づき:** 人は、未来の大きなご褒美より、今の小さなご褒美を大切にしてしまうことがある。<br>**完成(パート2完了時):**<br>・**やってみたこと:** 雨の日に、将来の割引じゃなくて、その場ですぐもらえるプレゼントを提供してみた。<br>・**わかったこと:** 雨の日でもお客さんが来てくれるようになった。<br>・**自分だけの法則:** 今すぐもらえる喜びの法則 -                                                              |
| **報酬**           | +120 評判ポイント、内装アイテム「雨音の聞こえる窓」 -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
