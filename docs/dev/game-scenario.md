# ゲームシナリオ設計書

> **【このドキュメントの役割】**
> このドキュメントは、[世界観・ストーリー(story.md)](story.md)で定義された物語を、具体的なゲームのチャプター、解放条件、体験の構造（レベルデザイン）に落とし込むことを目的とします。
> 個々のセリフ集ではなく、ゲームの進行と体験の構造を定義します。

## 第一部：問いと仮説、最初の発見

### Chapter 1: 三つの選択肢

| 項目                    | 内容                                                                                                                                                                                                                  |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**            | 序章クリア                                                                                                                                                                                                            |
| **発見される概念**      | **おとり効果 (Decoy Effect)**                                                                                                                                                                                         |
| **1. 物語イベント**     | カフェのメニューを試しに「安いブレンド」「高いブレンド」の2種類にしてみる。しかしお客さんは安い方ばかりを頼み、高い方は敬遠されがち。                                                                                 |
| **2. 教授の問い**       | 夕方、店を訪れたホーソーン教授が静かにつぶやく。「人間とは面白いものだ。二つのものしか知らないと、つい比較して安い方を選んでしまう。だが、三つ目の選択肢がそこにあったなら…彼らの目は何に安らぎを見出すのだろうね？」 |
| **3. プレイヤーの仮説** | ニャッジとの壁打ち会話。「うーん、高いのはみんな怖いのかなあ？」「真ん中があったら安心するのかニャ？」。プレイヤーは「極端な選択肢を避けるのでは？」という仮説を立てる。                                              |
| **4. 検証フェーズ**     | 1. メニューを「松」「竹」「梅」の3種類にする。<br>2. 真ん中の価格の「竹」が一定数売れる。                                                                                                                             |
| **5. 発見ノート記録**   | **WHY:** なぜ人は真ん中を選ぶ？<br>**HOW:** メニューを3種類にしてみた。<br>**WHAT:** 真ん中のメニューが一番人気になった。<br>**MEMO:** 人は極端な選択を避ける傾向があるみたい。                                       |
| **報酬**                | +50 評判ポイント、キーアイテム「アンティークなメニューブック」                                                                                                                                                        |

### Chapter 2: 最初の印象

| 項目                    | 内容 -                                                                                                                                                                                                 |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **解放条件**            | Chapter 1 クリア、評判が50以上 -                                                                                                                                                                       |
| **発見される概念**      | **アンカリング効果 (Anchoring Effect)** -                                                                                                                                                              |
| **1. 物語イベント**     | 街で評判のパン屋が、目玉商品として非常に高価な「黄金のクロワッsan」を数量限定で売り出し、話題になっている。 -                                                                                          |
| **2. 教授の問い**       | ホーソーン教授が、パン屋の袋を片手にカフェを訪れる。「あのクロワッサン自体を買う人は少ないだろう。だが、店の他のパンは、なんだかいつもより魅力的に見えなかったかね？…不思議なことだ。」 -              |
| **3. プレイヤーの仮説** | 「最初にすごく高いものを見ると、他のものが安く見えるのかも？」。ニャッジは「ニャーは1000円のカリカリを見た後だと、500円のカリカリが安く見えちゃうニャ」と同意する。 -                                  |
| **4. 検証フェーズ**     | 1. 高価格な「限定スペシャルケーキ」を開発し、メニューの一番上に配置する。<br>2. 1日の売上が目標額を達成する。 -                                                                                        |
| **5. 発見ノート記録**   | **WHY:** なぜ高い商品があると他が売れる？<br>**HOW:** 高価な限定ケーキを最初に表示した。<br>**WHAT:** 他の商品の売上も上がった。<br>**MEMO:** 人は最初に見た数字（アンカー）に強く影響されるみたい。 - |
| **報酬**                | +70 評判ポイント、内装アイテム「本日のおすすめ用イーゼル」 -                                                                                                                                           |

### Chapter 3: 言葉の額縁

| 項目                    | 内容 -                                                                                                                                                                                         |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **解放条件**            | Chapter 2 クリア、評判が150以上 -                                                                                                                                                              |
| **発見される概念**      | **フレーミング効果 (Framing Effect)** -                                                                                                                                                        |
| **1. 物語イベント**     | ヘルシーさが売りのコーヒーを開発したが、「脂肪分10%」という正直な表記が逆に不健康な印象を与えているのか、売れ行きが良くない。 -                                                                |
| **2. 教授の問い**       | 「真実を伝えることは大切だ。だが、同じ絵でも、飾る額縁の色によって悲しい絵が楽しい絵に見えることもある。君の言葉は、どんな額縁に入っているのだろうね？」 -                                     |
| **3. プレイヤーの仮説** | 「『脂肪分10%』はネガティブに聞こえるのかも。『脂肪分90%カット』なら、ポジティブに聞こえるかも？」 -                                                                                           |
| **4. 検証フェーズ**     | 1. 同じコーヒーの紹介文を、よりポジティブな表現（例：「脂肪分90%カット」）に変更する。<br>2. 変更後のメニューの注文数が、変更前の2倍になる。 -                                                 |
| **5. 発見ノート記録**   | **WHY:** なぜ表現を変えると売上が変わる？<br>**HOW:** 同じ内容をポジティブな表現に変えた。<br>**WHAT:** 注文が増えた。<br>**MEMO:** 人は同じものでも、伝え方次第で印象が大きく変わるみたい。 - |
| **報酬**                | +100 評判ポイント、レシピ「言葉を飾るシュガー」 -                                                                                                                                              |

## 第二部：すれ違う心と、雨の日のコーヒー

### Chapter 4: 今日の喜び

| 項目                    | 内容 -                                                                                                                                                                                                             |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **解放条件**            | 第二部開始 -                                                                                                                                                                                                       |
| **発見される概念**      | **現在志向バイアス (Present Bias)** -                                                                                                                                                                              |
| -                       |
| **1. 物語イベント**     | 雨の日になると、客足がぱったりと途絶える。将来お得になるポイントカードを導入してみたが、効果がない。 -                                                                                                             |
| **2. 教授の問い**       | 雨宿りに訪れたホーソーン教授が語る。「未来の大きな虹を約束されるより、今日の小さな傘の方が、人の心を温めることもある。君のお客様は、『今』何を求めているのだろうね？」 -                                           |
| **3. プレイヤーの仮説** | 「将来の大きな得より、すぐ目の前の小さな喜びが大切なのかも。」 -                                                                                                                                                   |
| **4. 検証フェーズ**     | 1. 「雨の日限定！ケーキもう一個プレゼント」のような、即時的な喜びを提供する施策を試す。<br>2. 雨の日の客数が増加する。 -                                                                                           |
| **5. 発見ノート記録**   | **WHY:** なぜポイントカードは効かなかった？<br>**HOW:** すぐにもらえる喜びを提供した。<br>**WHAT:** 雨の日でもお客さんが来てくれた。<br>**MEMO:** 人は未来の大きな得より、今の小さな喜びを大切にすることがある。 - |
| **報酬**                | +120 評判ポイント、内装アイテム「雨音の聞こえる窓」 -                                                                                                                                                              |
